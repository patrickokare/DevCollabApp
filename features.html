Implementation Deadline August 8th 2016

Project Specification in a Nutshell...........brainstorming

Project Name KareTech pronounced as ArrayTech; the idea was gotten from Arrays in programming which can take multiple items,
the Application should have multiple Features(Collaborative feature) that will help in a Collaborative Learning Environment.
Key Features are; A Code Editor, Audio/Video Conferencing, mobile framework,file Upload Option and Text Chatting in real-time

  • The Application Must have a Code editor(Code mirror or Any other)Something Similar CodeShare Editor
  • The Application must make use of Google Hangout Audio and Video Conferencing in Real-time.
  • The Application Must make use of bootstrap Framework to make it responsive on multiple devices
  • Chatting in Real-time similar to Facebook Messenger
  • The Application must be downloadable from an App Store irrespective of the Mobile OS using PhoneGap as a Native wrapper- similar to the way Instagram or Facebook Mobile App works
  • The Application Must make use of any file Upload API's BUT NOT GOOGLE DRIVE API- using the File Upload Function in PHP if Possible.
  • Makes use of Azure Cloud-based Services, GitHub
  • Optional- More Interactive and Dynamic UI desgin and Color Scheme which will make it more Appealing to Mobile Users.
